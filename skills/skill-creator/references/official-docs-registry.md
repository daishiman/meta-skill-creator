ドキュメント生成時に最新の公式情報を参照するために使用。

## 使用方法

ドキュメント生成時、対象APIの公式ドキュメントURLを取得し、WebSearch/WebFetchで最新情報を確認する。

```javascript
// 使用例
const provider = "google";
const api = "gmail";
const docs = registry[provider][api];
// → { quickstart: "https://...", reference: "https://..." }
```
